# Planificador Inteligente de Hotel - Isla Paraíso

## Descripción

Planificador de eventos para un hotel ficticio en una isla tropical.  
Gestiona **eventos**, **recursos limitados** y **restricciones**, evitando conflictos de horarios y recursos.  

Actualmente funciona en **Python** con un core básico; más adelante se añadirá GUI con CustomTkinter.

---

## Instalación

# 1. Clonar el repositorio:
```bash
git clone https://github.com/tu_usuario/mi_proyecto_hotel.git
cd Hotel_Planner_Project
```

# 2. Crear y activar el entorno virtual:
```bash
python3 -m venv venv
source venv/bin/activate   # Linux/Mac
# .\venv\Scripts\activate # Windows
```

# 3. Instalar dependencias:
```bash
pip install -r requirements.txt
```

## Funcionalidades Planeadas

1. Evitar conflictos de recursos al planificar eventos.

2. Validar restricciones entre recursos.

3. Manejar eventos recurrentes (diarios, semanales, mensuales).

4. Buscar automáticamente horarios disponibles.

5. Guardar y cargar datos desde archivos JSON.

6. Interfaz gráfica con CustomTkinter (fase futura).
